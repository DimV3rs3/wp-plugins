# Общий сбор всех плагинов для World Statistics Platform

!!! В данном виде плагины пересобраны и приведены к виду Плагин+Эргономичность раздельно !!!
Данные файлы имеют приписку .mod, в названиях они тоже указаны. Эти плагины можно удалять/не использовать.
Плагины без приписки .mod - самые последние доступные версии плагинов без притрагивания к коду.

---

Основной плагин - https://github.com/DimV3rs3/world-statistics-platform

Плагин эргономичности - https://github.com/DimV3rs3/worldstat-ergonomics

---

# Установка
```
git clone https://github.com/DimV3rs3/wp-plugins.git plugins
git clone https://github.com/DimV3rs3/world-statistics-platform.git plugins/world-statistics-platform
git clone https://github.com/DimV3rs3/worldstat-ergonomics.git plugins/worldstat-ergonomics
echo "Все репозитории успешно склонированы."
```
